# Cookbook

Receitas que combinam múltiplos domínios da plataforma para resolver problemas analíticos reais. Cada receita é auto-contida, com código executável e referências cruzadas aos pacotes/conceitos relevantes.

O Cookbook é o lugar onde **Modularidade** sai do princípio abstrato para a prática: ferramentas independentes, sem acoplamento, compostas em pipelines analíticos coerentes.

## Receitas

- **[Análise Econômica Multi-Fonte](analise-economica-multi-fonte.md)** — cruzar IPCA (IBGE) com yields do Tesouro e indicadores de emprego (RAIS) para construir um painel macroeconômico básico. Demonstra `sidra-fetcher` + `tddata` + `pdet-data` em um único fluxo.

Mais receitas serão adicionadas ao longo do tempo (Saúde + Clima, Comércio + Trabalho, etc.). Sugestões: abra issue no repositório.
