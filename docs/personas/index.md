# Quem usa Quantilica

Esta seção é um atalho. Você descreve o seu trabalho; a página devolve as três ferramentas mais úteis, a receita do Cookbook que faz sentido começar, e o conceito que vale ler antes.

## Por perfil

- **[Analista macroeconômico](analista-macro.md)** — séries, indicadores, painéis e relatórios. IBGE, BCB, Tesouro.
- **[Cientista de dados de saúde pública](cientista-saude.md)** — microdados, vigilância, mortalidade, vacinação. DATASUS + IBGE.
- **[Engenheiro de dados / ETL](engenheiro-dados.md)** — pipelines, warehouse, contracts, proveniência. Toda a stack.
- **[Pesquisador acadêmico](pesquisador.md)** — reprodutibilidade, séries longas, snapshots versionados.

Nenhum dos perfis cobre o seu? Comece pelo [Quickstart](../quickstart.md) e pelos [Princípios de Design](../concepts/principios.md) — eles funcionam para qualquer caso de uso analítico.
